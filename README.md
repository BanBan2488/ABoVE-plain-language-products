# ABoVE-plain-language-products
NASA ABoVE (Arctic-Boreal Vulnerability Experiment) GIS mapping project integrating multiple NASA NCCS climate and ecosystem model datasets to analyze environmental conditions across Arctic-Boreal regions. This work includes the development of regional maps and plain-language booklet products for areas such as the Yukon-Kuskokwim Delta, Vuntut Gwitchin Traditional Territory, Fairbanks, Dehcho, Bristol Bay, and Sahtu, with additional regions including Whitehorse and the Mackenzie Delta in progress. The project focuses on translating complex geospatial data into accessible, community-centered visualizations.
